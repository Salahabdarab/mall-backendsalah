# Mall Backend (NestJS + TypeORM + Postgres)

## Quick Start (Replit/Atoms)
1) Upload this folder to Replit/Atoms (Node.js)
2) Add a Postgres database and set `DATABASE_URL`
3) Copy `.env.example` to `.env` and adjust
4) Run:
```bash
npm install
npm run start:dev
```
